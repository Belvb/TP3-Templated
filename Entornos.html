Bienvenido a Nuestra Página

"sembrando valores para cosechar virtudes"

Este es el Instituto Mariano Moreno, una escuela agropecuaria con diferentes entornos formativos para el alumno donde conocerá como funciona la producción de las diferentes especies como pollos parrilleros,gallinas ponedoras, porcinos, conejos, avejas, ganado vacuno, sector huerta y vivero. Tambien luego de completar los 7 años de escolaridad se le otorgará junto con el titulo vachiller orientado a ciencias naturales el titulo de Tecnico Agropecuario.
